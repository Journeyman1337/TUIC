TuiPanel Struct Reference

TUI rendering panel. This is a framebuffer for rendering TuiBatch objects and other TuiPanel objects to. These objects should only be used on the same thread as its TuiInstance. This object is opaque, and its members are not meant to be accessed directly unless implementing a backend. Use the functions in panel.h instead.
